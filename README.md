# covid_19

A basic application for tracking the data of COVID-19. Though the UI is quite bad, but hope I can upgrade the UI next time.


  # Features
   - *Real Time Search*
   - *All Countries data*
   - *Theme switch*
   - *Pie - Chart*
   - *Today and Alltime data showing*
   - *Some FAQs*
   - *Most affected country list*
